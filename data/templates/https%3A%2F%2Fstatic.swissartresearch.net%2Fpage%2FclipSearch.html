[[i18n "searchAI" bundle="bso"]]

Enter a description for an image and the system will try to find an image in the BSO collection that matches the description.


This search uses CLIP, an artificial neural network trained on a very large collection of images and their captions. Using CLIP, it is possible to calculate the similarity between a text and images. The search returns the images from the collection that match the entered text most closely.

The search always returns results, even when the collection does not contain any image that closely matches the search text. Therefore the search will work better for things that actually appear in the collection (e.g. 'a mountain', 'a group of people', 'sailboats on a lake') than for things that are unlikely to be present (e.g. 'a spaceship', 'brutalist architecture').

As CLIP has been trained on english captions, the search will work better for english text than for other languages.
